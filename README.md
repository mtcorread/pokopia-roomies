# Pokopia Roomies

A housing optimizer for **Pokemon Pokopia** on Nintendo Switch 2.

Find the best housing groups for your Pokemon based on shared favorite categories, habitat compatibility, and evolution line.

**[Open the app](https://mtcorread.github.io/pokopia-roomies/)**

## Features

- **Plan World** — Auto-assign all your Pokemon in a world into optimal house groups
- **Find Groups** — Search for the most compatible roommate groups of any size
- **My Pokemon** — Track which Pokemon you own and assign them to worlds
- **Compare** — See shared and unique categories between Pokemon
- **Categories** — Browse all favorite categories and the Pokemon that like them
- **Import/Export** — Save and load your collection as a JSON file

## Data Source

Pokemon data sourced from [Serebii.net](https://www.serebii.net/pokemonpokopia/favorites.shtml).
